### 5.3 Vulnérabilité principale

#### 🟠 [HIGH] Validation tenant sur EnergyConfiguration
**CWE** : CWE-639 (Authorization Bypass Through User-Controlled Key)

**Problème** : `findByCountryCode()` ne filtre pas par `tenantId`. Si les configurations sont tenant-scoped, risque de fuite de données business.

**Code concerné** :
```java
// Ligne 51-53
final EnergyConfiguration configuration = energyConfigurationProvider
        .findByCountryCode(normalizeCountryCode(countryCode))
        .orElseThrow(() -> new EnergyConfigurationNotFoundException(countryCode));
```

**Recommandation** :
```java
energyConfigurationProvider
    .findByTenantIdAndCountryCode(asset.getTenantId(), normalizeCountryCode(countryCode))
```

### 5.4 Autres vulnérabilités
| Sévérité | Issue | Action |
|----------|-------|--------|
| 🟡 Medium | Format countryCode non validé | Regex ISO 3166-1 alpha-2 |
| 🟡 Medium | BigDecimal non validés après calcul | Ajouter checks null/négatif |
| 🟡 Medium | Log injection via countryCode | Sanitizer avant exception |
| 🟢 Low | Instant.now() non injectable | Injecter Clock |
| 🟢 Low | usageDurationDays sans limite max | Limiter à 3650 jours |

### 5.5 Points positifs sécurité
- ✅ Classe `final` (anti-extension malveillante)
- ✅ `BigDecimal` pour précision numérique
- ✅ `UUID.randomUUID()` cryptographiquement sûr
- ✅ Normalisation input (`trim().toUpperCase(Locale.ROOT)`)
- ✅ Exceptions custom (pas de stack traces exposées)
- ✅ `tenantId` propagé dans `CarbonFootprint`

## 📈 7. Tests Unitaires Suggérés

### Structure de test recommandée

```java
@ExtendWith(MockitoExtension.class)
class CarbonCalculationEngineTest {

    @Mock CarbonCalculationStrategyFactory strategyFactory;
    @Mock EnergyConfigurationProvider energyConfigurationProvider;
    @Mock CountryEnergyCoefficientPolicy countryEnergyCoefficientPolicy;
    @Mock CarbonCalculationStrategy strategy;
    @Mock Clock clock;
    
    CarbonCalculationEngine engine;
    
    @BeforeEach
    void setUp() {
        engine = new CarbonCalculationEngine(
            strategyFactory, 
            energyConfigurationProvider, 
            countryEnergyCoefficientPolicy,
            clock
        );
    }
    
    // === HAPPY PATH ===
    
    @Test
    void shouldCalculateFootprintForValidAsset() { /* ... */ }
    
    @Test
    void shouldNormalizeCountryCodeToUppercase() { /* ... */ }
    
    @Test
    void shouldApplyCountryCoefficientCorrectly() { /* ... */ }
    
    @Test
    void shouldReturnFootprintWithCorrectScale() { /* ... */ }
    
    // === EDGE CASES ===
    
    @Test
    void shouldHandleCountryCodeWithWhitespace() { /* ... */ }
    
    @Test
    void shouldCalculateForMinimalUsageDuration() { /* ... */ }
    
    @Test
    void shouldHandleZeroEnergyConsumption() { /* ... */ }
    
    // === ERROR CASES ===
    
    @Test
    void shouldThrowWhenAssetIsNull() {
        assertThrows(NullPointerException.class, 
            () -> engine.calculateCarbonFootprint(null, 30, "FR"));
    }
    
    @Test
    void shouldThrowWhenUsageDaysIsZeroOrNegative() {
        assertThrows(CalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 0, "FR"));
        assertThrows(CalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, -1, "FR"));
    }
    
    @Test
    void shouldThrowWhenCountryCodeIsNullOrBlank() {
        assertThrows(CalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 30, null));
        assertThrows(CalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 30, "   "));
    }
    
    @Test
    void shouldThrowWhenCountryConfigurationNotFound() {
        when(energyConfigurationProvider.findByCountryCode("XX"))
            .thenReturn(Optional.empty());
        assertThrows(EnergyConfigurationNotFoundException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 30, "XX"));
    }
    
    @Test
    void shouldWrapUnexpectedRuntimeException() {
        when(strategy.computeEnergyConsumptionKwh(any(), anyInt()))
            .thenThrow(new RuntimeException("Unexpected"));
        CalculationException ex = assertThrows(CalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 30, "FR"));
        assertThat(ex.getCause()).isInstanceOf(RuntimeException.class);
    }
    
    @Test
    void shouldPropagateAssetCalculationException() {
        when(strategy.computeEnergyConsumptionKwh(any(), anyInt()))
            .thenThrow(new AssetCalculationException("Asset error"));
        assertThrows(AssetCalculationException.class, 
            () -> engine.calculateCarbonFootprint(validAsset, 30, "FR"));
    }
    
    // === CONSTRUCTOR VALIDATION ===
    
    @Test
    void shouldRejectNullDependencies() {
        assertThrows(NullPointerException.class, 
            () -> new CarbonCalculationEngine(null, energyConfigurationProvider, countryEnergyCoefficientPolicy));
        assertThrows(NullPointerException.class, 
            () -> new CarbonCalculationEngine(strategyFactory, null, countryEnergyCoefficientPolicy));
        assertThrows(NullPointerException.class, 
            () -> new CarbonCalculationEngine(strategyFactory, energyConfigurationProvider, null));
    }
}
```
